# BookWormApp

BookWormApp is a web application that allows users to search for books, browse reviews, and share their own opinions on books they have read. The platform provides an accessible way for readers to discover new books and engage with a community of book enthusiasts.

## Tech stack

- Node.js (Express)
- React (Typescript)
- PostgreSQL

## My contribution

- Collaborated in a team environment on a full-stack web application
- Developed backend logic and REST API endpoints using Node.js
- Designed and implemented PostgreSQL database schema
- Worked on system features and application logic

## Features

The application enables users to search for books and view reviews submitted by other users. Registered users can create accounts, log in, and write their own reviews for selected books. Visitors can access and browse the platform without an account, but reviewing functionality is restricted to authenticated users.
